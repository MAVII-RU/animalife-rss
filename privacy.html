Сайт

Сайт animalife (страницы на mavii-ru.github.io/animalife-rss) — это редакционный ресурс медиа AnimaLife. Сайт не собирает персональных данных посетителей: нет форм регистрации, нет аналитических куки, нет рекламных трекеров.

Хостинг

Сайт размещён на платформе GitHub Pages. При посещении страниц GitHub может собирать стандартные технические данные (IP-адрес, User-Agent, время запроса) для обеспечения работы сервиса. Подробнее: GitHub Privacy Statement.

Внешние ссылки

Сайт содержит ссылки на ресурсы AnimaLife: Telegram-бот, MAX-бот, страницу в Дзене. Использование этих ресурсов регулируется политиками соответствующих платформ.

Обработка данных в приложении

Если вы переходите по ссылке в Telegram-бот или MAX-бот AnimaLife и пользуетесь приложением — обработка ваших данных регулируется политикой конфиденциальности приложения, доступной внутри бота.
